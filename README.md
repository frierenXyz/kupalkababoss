# 🔐 suicideGF - CN31 Token Fetcher API

High-performance NetEase Captcha (Dun163) Token Generator Service

## ✨ Features

- 🚀 High-performance async API server
- 🔄 Automatic token caching
- 📦 Batch token generation
- 🔧 Easy deployment anywhere

```bash
# Clone
git clone https://github.com/YOUR_USERNAME/suicideGF.git
